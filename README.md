# 한국 추계인구 데이터 분석
한국 추계인구 데이터를 분석하고, 이를 바탕으로 인구 변화 및 예측 분석을 수행하는 프로젝트입니다. 주어진 데이터를 활용하여 한국의 인구 동향을 시각화하고, 향후 인구 예측 모델을 생성합니다.

## 선택이유
한국의 추계인구 데이터를 바탕으로 인구 변화 분석 및 미래 예측 데이터는 주로 정부 기관인 통계청에서 제공하는 자료를 활용하며, 이 분석을 통해 향후 인구 구조 변화에 출산,고령,성별,연령등의 관점을 볼수 있습니다

## 분석 내용
사용된 데이터는 주로 한국의 추계인구 데이터로, 연도별, 성별, 연령대별 인구 수를 포함하고 있습니다. 주요 컬럼은 다음과 같습니다
연도: 데이터가 측정된 연도
성별: 남성, 여성으로 구분된 인구 수
연령대: 0-4세, 5-9세, 10-14세 등으로 구분된 연령대
인구수: 해당 연령대와 성별에 따른 인구 수
성별/연령대별 분포 분석: 성별 및 연령대별 인구 변화를 분석하여 미래의 사회적 변화에 대한 예측을 제공합니다.
시계열 데이터 시각화하여 연도별 인구 변화, 성별 및 연령대별 인구 수를 시각화하여 추세를 파악합니다.
인구 예측 데이터로 향후 몇 년간의 한국 인구 변화를 예측합니다.
